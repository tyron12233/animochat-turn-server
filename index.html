Welcome to the P2P Test!

We're testing a new Peer-to-Peer (P2P) chat system. This is a big step away from using a central server like Supabase for messaging.

Why the change?

  • Better Maintainability: Simpler infrastructure for the future.
  • Enhanced Privacy: Messages go directly between users.
  • Faster Connections: Less server delay means quicker messages.
What is Peer-to-Peer?

Peer-to-Peer (P2P) chat means your messages go directly to the other user without passing through a central server. This provides better privacy and can be faster. Our server only helps to "introduce" two users to each other.
